App Name
Event-Look-Up
Description
This is an add-on app for event publication. Vendors signup to push their up coming event. It is a part of the coming mega advertising app which is projected to showcase different events, ranging from business meeting, sales exstravaganza, sporting events, entertainment, trade exhibitions, schools, church and other social events meant for public consumption.
